Air Quality in India - Interactive Project

### Simple interactive tool that displays the air quality in cigarettes and particulate matter of different Indian cities. 

INTRODUCTION
------------

This interactive project allows users to select an Indian city and get the local air quality of that city displayed in cigarettes and particulate matter. 

The project presents each city in the form of buttons, which can be clicked by users to have the air quality displayed on the right. 

There is a version in English and a version in Hindi. 

TECHNOLOGIES
------------

Project is created with: 
. HTML
. CSS
. Javascript

LAUNCH
------------
To run this project: 

. In the ./dist folder, you will find the two versions of the project: english and hindi version.

. In the ./src folder, you will find two sets of data: one presenting the data in English and one presenting the data in Hindi. However, in the project, the JSON data is pulled from a public URL: https://github.com/bbc/vj-code-tests/tree/master/airquality/src/data.
